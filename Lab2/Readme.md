# Лабораторная работа: Сборка сайта с использованием Node.js, Vite и Bootstrap

## Описание работы

В рамках данной лабораторной работы был создан простой сайт с использованием современных инструментов сборки веб-проекта.  
Основной целью работы было изучение процесса сборки сайта с помощью **Node.js**, **npm**, **Vite** и препроцессора **SCSS**, а также использование CSS-фреймворка **Bootstrap** для создания пользовательского интерфейса.

Готовый результат проекта опубликован с помощью **GitHub Pages**, что позволяет просматривать сайт напрямую через браузер.

## Демо проекта

Сайт доступен по ссылке:

https://bluv4nul.github.io/CompPractice/

## Используемые технологии

В проекте были использованы следующие технологии:

- Node.js
- npm
- Vite (сборщик проекта)
- Bootstrap 5
- SCSS
- GitHub Pages

## Структура проекта

Проект имеет следующую структуру:

```
lab2
├ src
│ ├ js
│ │ └ main.js
│ ├ scss
│ │ └ styles.scss
│ └ index.html
├ dist
├ package.json
└ vite.config.js
```

### Описание основных файлов

**index.html**

Основная HTML-страница сайта.  
Содержит структуру интерфейса и подключение собранных ресурсов.

**main.js**

Главный JavaScript файл проекта.  
Используется для подключения Bootstrap и пользовательских стилей.

**styles.scss**

Файл пользовательских стилей, в котором добавлены:

- оформление главного блока
- стили секций
- оформление карточек
- hover-эффекты
- оформление footer

**vite.config.js**

Конфигурация сборщика Vite.  
Используется для настройки:

- базового пути проекта
- корневой директории
- папки сборки

## Реализованный интерфейс

На странице реализованы следующие элементы интерфейса:

- навигационная панель
- главный информационный блок
- секция с описанием проекта
- карточки с примерами компонентов
- кнопки Bootstrap
- модальное окно
- footer

Для построения сетки и компонентов использовался **Bootstrap**.

## Использование SCSS

SCSS использовался для добавления пользовательских стилей поверх Bootstrap.

Были реализованы:

- фон главного блока
- отступы секций
- стили карточек
- hover-эффекты
- оформление footer

## Сборка проекта

Для запуска проекта используется Vite.

### Установка зависимостей

```
npm install
```

### Запуск проекта

```
npm run dev
```

### Сборка проекта

```
npm run build
```

## Публикация проекта

Сайт был опубликован с помощью **GitHub Pages**.

Для деплоя используется пакет **gh-pages**.

Команда публикации:

```
npm run deploy
```

После выполнения команды содержимое папки `dist` автоматически публикуется в ветке `gh-pages`.
